# Slack clone using Django with Channels and HTML over the Wire architecture

- Messages in real time.
- Logic in the backend.
- Groups.
- Private messages.
- Login.
- Minimal use of JavaScript (35 lines without comments).

## Run 🏃

```
docker-compose up
```
